# TopGear

אפליקציית שולחן עבודה לניהול מוסך — יומן עבודות, מלאי חלקים ומסירות. עובדת **100% מקומית במחשב, ללא חיבור אינטרנט וללא שרת**.

---

## התקנה (Windows)

1. עברו ל[עמוד הגרסה האחרונה](../../releases/latest).
2. הורידו את הקובץ **`TopGear_…_x64-setup.exe`**.
3. הריצו את ההתקנה.
   אם Windows SmartScreen מציג התרעה (האפליקציה אינה חתומה דיגיטלית) — לחצו **"More info"** ולאחר מכן **"Run anyway"**.
4. הפעילו את **TopGear** מתפריט ההתחלה.

זהו — אין הרשמה, אין שרת, אין חיבור אינטרנט.

## עדכונים אוטומטיים

המערכת בודקת בכל פתיחה אם פורסמה גרסה חדשה. אם כן, היא מציגה הודעה בעברית עם מספר הגרסה ושואלת אם להתקין. בלחיצה על "אישור" — ההורדה, ההתקנה והאתחול קורים אוטומטית, **ללא איבוד נתונים**.

## יכולות עיקריות

- **יומן עבודה** בעברית וב-RTL מלא, עם חישוב אוטומטי של יום בשבוע, סה"כ ורווח.
- **קטלוג מלאי** עם מק"ט ייחודי, כמות, מחיר למוסך ומחיר ללקוח.
- **קטגוריות מותאמות אישית** — הוסיפו קטגוריות חדשות בלחיצה אחת.
- **בחירת חלקים בעבודה** מתוך כרטיסיות, עם תצוגת מלאי שמתעדכנת בזמן אמת בעת הבחירה.
- **יצירת חלק חדש** ישירות מתוך מסך הוספת עבודה.
- **ניכוי מלאי אוטומטי** בעת שמירת עבודה; **החזרת מלאי** בעת מחיקה.
- **מסך מסירות צפויות** עם סינון לפי טווח (היום / השבוע / החודש / באיחור).
- **באנר אנליטיקה** להיום, השבוע, החודש או הכל — הכנסות, עלויות ורווח.
- **ייצוא JSON מלא** וייצוא CSV ליומן ולמלאי; ייבוא מגיבוי JSON.

## קיצורי מקלדת

| מקש | פעולה |
|---|---|
| `Ctrl + N` | הוספת עבודה או חלק, לפי המסך הפעיל |
| `Esc` | סגירת חלון פתוח |
| `↑` `↓` | ניווט בשורות הטבלה (כשהפוקוס אינו בתוך שדה) |

## פרטיות ואחסון נתונים

- **כל הנתונים נשארים במחשב.** אין שרת, אין סנכרון, אין טלמטריה.
- הנתונים נשמרים תחת `%LOCALAPPDATA%\com.topgear.garage\` בפרופיל המשתמש של Windows.
- לגיבוי / העברה בין מחשבים — השתמשו ב**"ייצוא JSON מלא"** ממסך **גיבוי וייצוא**, ושמרו את הקובץ במקום בטוח.
- עדכוני גרסה אינם מוחקים את הנתונים — `%LOCALAPPDATA%` נשמר.

## הפעלה ללא התקנה (אופציונלי)

ניתן גם לפתוח את `index.html` בדפדפן שולחני מודרני (Chrome / Edge). הנתונים יישמרו ב-IndexedDB של הדפדפן (תלוי-פרופיל). מצב זה אינו מקבל עדכונים אוטומטיים.
